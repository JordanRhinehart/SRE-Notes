	==> STANDARDIZATION: CONTAINERS <==
	
	-> Thin packaging model - new type of VM
	-> Packages software, its dependencies and configuration files
	-> Can be copied between machines
	-> Uses the underlying OS
	
	
	Containers vs VM
	 
	Containers are more portable
	Containers do not have OS
	Container is everything above the operating system
